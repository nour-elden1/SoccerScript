# SoccerScript Lexical Analyzer

This project is a Python-based lexical analyzer for SoccerScript, a soccer-themed programming language. The analyzer scans and tokenizes SoccerScript code into logical units (tokens) such as keywords, identifiers, numbers, strings, symbols, and comments.

## How to Run
1. Make sure Python is installed on your system.
2. Run the `index.py` file:
   ```bash
   python index.py
